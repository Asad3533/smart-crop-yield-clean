# 🌾 Smart Crop Yield Prediction System – Machine Learning & Data Science

An AI-powered Crop Yield Prediction System built using Machine Learning, Streamlit, and Agricultural Data Analytics to predict crop production based on environmental and farming factors such as rainfall, temperature, pesticide usage, crop type, and geographical location.

---

# 🚀 Project Overview

The Smart Crop Yield Prediction System leverages historical agricultural data and machine learning algorithms to estimate future crop yields.

The project combines data from multiple agricultural sources, including rainfall, temperature, pesticide usage, and crop production datasets, to provide data-driven insights for smart farming and agricultural planning.

The application enables users to input farming conditions and receive real-time crop yield predictions through an interactive Streamlit dashboard.

The trained machine learning model is not included in this repository because it exceeds GitHub's file size limit.

Download the model from:

https://drive.google.com/file/d/1Yx_Ew5qDdLjmk7mGb3tcGnCKdILBS_pS/view?usp=sharing

After downloading:

1. Place `crop_yield_model.pkl` in the project root directory.
2. Ensure it is in the same folder as `app.py`.
3. Run the Streamlit application.

# ▶️ Running the Project

## Clone Repository

```bash
git clone https://github.com/Asad3533/smart-crop-yield-clean.git
cd smart-crop-yield-clean
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Streamlit Application

```bash
streamlit run app.py
```
